# Система управления электронной очередью (Т-банк)

**WEB-DEV Track (оригинальный таск)**:
Реализованная программа должна соответствовать тербованиям:
✅ Для участников:
Позволяет занять место в очереди удаленно
Показывает примерное время ожидания
Дает возможность временно отлучиться, не теряя место
Уведомляет о приближении очереди
✅ Для организаторов:
Четко отображает порядок участников
Позволяет управлять очередью (пропускать, добавлять)
Снижает нагрузку на персонал стенда
